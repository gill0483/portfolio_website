Astra Album

Oct 2021 School Project

Astra Album is website where we sell music albums worldwide. You can search any albums and order which is available in our store by adding in cart, you can also signup and save favourite albums to order later by login.

In this project, we mainly used HTML, CSS, JavaScript and Bootstrap(for frame work). We divide this project in 3 parts as we was 3 members in the group, I worked on search page to implement search feature according to key words.

Snap-Shots

Below you can see some snapshots of our website to get idea how I our website look like.

1. Landing Page

This is the main page with navigation bar which incudes search feature, signup and cart option.

2. Search Page

Once we click on search button it will take to search page and it will show all albums that include searched key word in Arraylist.

3. Product Page

Product page will open with information of the album that is clicked and will give feature to add the album to cart, check description and reviews.
